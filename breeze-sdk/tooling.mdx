## Breeze Decoder

A decoder for Breeze on-chain state and instructions, similar to how Carbon decoder works but built specifically for Breeze.

<Card title="Breeze Decoder" icon="code" href="https://github.com/anagrambuild/breeze-decoder">
  Decode Breeze on-chain state and instructions for easier integration and debugging.
</Card>

Use the decoder to:
- Parse Breeze account data
- Decode transaction instructions
- Debug on-chain interactions

```bash
# Clone the repository
git clone https://github.com/anagrambuild/breeze-decoder
```

## Breeze CPI Crate

We provide a CPI (Cross-Program Invocation) crate for integrating Breeze directly into your on-chain Solana programs.

<Warning>
The Breeze CPI crate is available for advanced on-chain integrations. Please [contact us](/contact-us/contact-us) before using it to ensure proper setup and support.
</Warning>
