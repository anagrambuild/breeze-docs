The Breeze Admin Portal is a comprehensive web-based dashboard that provides a user-friendly interface for managing your Breeze integration. Built on top of the Admin API, the portal simplifies complex administrative tasks through an intuitive interface, making it easy to monitor funds, manage users, and oversee your entire Breeze deployment without requiring technical expertise.

Whether you're setting up new funds, monitoring performance metrics, or managing user access, the Admin Portal streamlines these operations with visual dashboards, automated workflows, and real-time insights that help you maintain optimal platform performance.
