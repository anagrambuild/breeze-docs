---
title: 'Installation'
description: 'Install Breeze UI for Your Next.js Project'
---

Run the init command to create a new Next.js project or to setup an existing one:

<CodeGroup>

```bash npm
npm i breeze-ui-library
```

```bash pnpm
pnpm add breeze-ui-library
```

```bash yarn
yarn add breeze-ui-library
```

```bash bun
bun add breeze-ui-library
```

</CodeGroup>

<Note>
Make sure you have Node.js 16+ installed before running the installation command.
</Note>

### CLI Installation

```bash
npx breeze-ui-library add simple_deposit
```

### Usage

<CodeGroup>

```tsx Basic Usage
import { SimpleDeposit } from "breeze-ui-library/client"

<SimpleDeposit />
```

```tsx Component Example
import { SimpleDeposit } from "breeze-ui-library/client"

export function SimpleDepositUI() {
  return <SimpleDeposit />
}
```

</CodeGroup>


<Check>You are good to go, explore different Breeze UI Components for Next.js framework</Check>
