# Introduction
The Model Context Protocol (MCP) is an open standard that links your functions to LLMs and AI applications. With Mintlify’s integration, you can effortlessly generate an MCP server for Breeze’s documentation, enabling seamless, AI-powered interactions with your product.

# Install Breeze MCP Server

Install the Breeze MCP server to enable AI-powered interactions with Breeze documentation. This allows LLMs and AI applications to seamlessly access and query Breeze docs through the Model Context Protocol.

```bash
npx mint-mcp add docs.breeze.baby
```

For detailed setup instructions and configuration options, check the [full MCP integration guide](https://mintlify.com/docs/mcp).
