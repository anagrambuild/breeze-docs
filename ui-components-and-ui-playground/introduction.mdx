## **What is Breeze UI Playground?**

Breeze UI Playground is a collection of pre-built, customizable UI components that enable developers to seamlessly integrate Breeze's DeFi yield strategies and risk management features into their applications. The playground provides an interactive environment where you can explore various components, test their functionality, and customize their appearance to fit your application's design.
