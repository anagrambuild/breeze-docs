<Steps>
  <Step title="Account Setup">
    <Card title="Create User Fund Account" icon="user-plus">
      Call `POST /create_user_fund/tx` (Breeze Admin API) with the user's wallet address and fund ID. This creates the necessary on-chain account for the user to participate in the fund. The user signs and submits this transaction to initialize their account.
    </Card>
  </Step>
  
  <Step title="Deposit Funds">
    <Card title="Process User Deposits" icon="arrow-down">
      When a user wants to deposit:
      - Use `POST /deposit/tx` to generate a complete transaction, or
      - Use `POST /deposit/ix` for more control over transaction construction
      - User signs the transaction with their wallet
      - Submit to Solana network to complete the deposit
    </Card>
  </Step>
  
  <Step title="Monitor Portfolio">
    <Card title="Track Performance & Analytics" icon="chart-line">
      Regularly update user portfolio data:
      - `GET /user/{user_id}/current_value` for real-time portfolio values
      - `GET /user/{user_id}/stats` for historical performance analytics
      - `GET /getFund/{fund_id}` for fund-level risk metrics and allocations
      - Display yield earned, current risk exposure, and portfolio growth
    </Card>
  </Step>

  <Step title="Withdraw Funds">
    <Card title="Process User Withdrawals" icon="arrow-up">
      When a user wants to withdraw:
      - Use `POST /withdraw/tx` to generate a complete withdrawal transaction, or  
      - Use `POST /withdraw/ix` for custom transaction handling
      - Specify number of shares to withdraw or use `"all": true` for full withdrawal
      - User signs the transaction to complete the withdrawal process
    </Card>
  </Step>
</Steps>
